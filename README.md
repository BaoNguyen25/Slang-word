# Slang Word - A dictionary application using Java Swing
## Features
```bash
• View Slang Word List
• Find Slang Based On Word, Definition
• View Search History
• Add, Edit, Delete A Slang Word
• Random A Slang Word
• Reset Original List
• Quiz Game
```
